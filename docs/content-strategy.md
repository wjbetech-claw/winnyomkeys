# Content strategy

## Brand / voice guidelines

The brand should feel:

- calm
- precise
- technically competent
- premium without sounding luxurious for its own sake
- honest rather than sales-heavy

## Messaging rules

Do:

- explain what is offered in plain language
- state Seoul, Korea clearly
- guide visitors toward the inquiry path
- set expectations without pretending certainty where there is none

Do not:

- invent testimonials
- invent certifications
- invent turnaround times
- invent pricing
- use hype-heavy startup language

## Copy placeholder policy

If business details are not confirmed, use explicit placeholders and label them clearly.
