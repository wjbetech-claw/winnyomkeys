# Design guidelines

## Desired feel

- premium
- minimal
- modern
- calm
- precise
- trustworthy

## Visual rules

- prioritize legibility over decoration
- keep spacing consistent and generous
- use restrained color accents
- avoid visual clutter and novelty effects
- maintain a clear CTA hierarchy

## Responsive rules

- design mobile-first
- preserve readability at smaller widths
- avoid horizontal overflow
- keep key calls to action visible without excessive scrolling

## Accessibility expectations

- semantic structure
- visible focus states
- sufficient contrast
- meaningful headings
- no text embedded in images for core content

## Motion rules

- animation should be subtle and purposeful
- avoid gimmicks, looping distractions, or noisy transitions
